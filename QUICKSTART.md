## Step 2: Configure Domain (5 minutes + 24-48 hours DNS)

### At Namecheap:
1. Log in to [namecheap.com](https://namecheap.com)
2. Go to "My Domains" ‚Üí "dhia-portfolio.me"
3. Click "Manage"
4. Go to "Nameservers"
5. Select "Custom DNS"
6. Enter these:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ns3.vercel-dns.com
   ```
7. Click Save

### At Vercel:
1. Go to your project dashboard
2. Click "Settings" ‚Üí "Domains"
3. Add domain: `dhia-portfolio.me`
4. Wait for "Verified" status (24-48 hours)

**Done! Your site is live at https://dhia-portfolio.me**

## FAQ

**Q: Will my site be ready immediately?**  
A: Your code deploys to Vercel immediately (~5 min), but DNS takes 24-48 hours for your domain to work.

**Q: How do I update my website?**  
A: Edit files in v0 (or locally), push to GitHub ‚Üí Vercel auto-deploys in ~1 minute.

**Q: What if something breaks?**  
A: With GitHub, you can rollback to any previous version instantly. No worries!

**Q: Do I need to know Git?**  
A: For first-time setup yes. After that, just `git add . && git commit -m "..." && git push`.

**Q: Can I keep using v0?**  
A: Yes! Continue developing in v0, then sync to GitHub when ready.
